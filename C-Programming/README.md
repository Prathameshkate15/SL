# C Programming

This directory contains fundamental C programming examples commonly asked in practical exams.

## Topics

1. **Basic Programs**
   - Hello World
   - Basic I/O operations
   - Variable types and operators

2. **Control Structures**
   - If-else statements
   - Switch-case
   - Loops (for, while, do-while)

3. **Functions**
   - Function definition and calling
   - Pass by value and reference
   - Recursion

4. **Arrays**
   - One-dimensional arrays
   - Two-dimensional arrays
   - String operations

5. **Pointers**
   - Pointer basics
   - Pointers and arrays
   - Dynamic memory allocation

6. **Structures**
   - Structure definition
   - Structure arrays
   - Nested structures

## Sample Programs

Check the individual `.c` files in this directory for complete programs with explanations.

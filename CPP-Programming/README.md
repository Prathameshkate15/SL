# C++ Programming

This directory contains C++ and Object-Oriented Programming examples for practical exams.

## Topics

1. **OOP Basics**
   - Classes and objects
   - Constructors and destructors
   - Access specifiers

2. **Inheritance**
   - Single inheritance
   - Multiple inheritance
   - Multilevel inheritance
   - Hierarchical inheritance

3. **Polymorphism**
   - Function overloading
   - Operator overloading
   - Virtual functions
   - Abstract classes

4. **Templates**
   - Function templates
   - Class templates

5. **Exception Handling**
   - Try-catch blocks
   - Custom exceptions

6. **STL Basics**
   - Vectors
   - Maps
   - Iterators

## Sample Programs

Check the individual `.cpp` files in this directory for complete programs with explanations.

# Data Structures

This directory contains implementations of various data structures commonly asked in practical exams.

## Topics

1. **Arrays**
   - Array operations
   - Searching (Linear, Binary)
   - Sorting (Bubble, Selection, Insertion, Quick, Merge)

2. **Linked Lists**
   - Singly linked list
   - Doubly linked list
   - Circular linked list

3. **Stacks**
   - Array implementation
   - Linked list implementation
   - Applications (expression evaluation, parenthesis matching)

4. **Queues**
   - Simple queue
   - Circular queue
   - Priority queue
   - Dequeue

5. **Trees**
   - Binary tree
   - Binary search tree
   - Tree traversals (Inorder, Preorder, Postorder)
   - AVL tree basics

6. **Graphs**
   - Graph representations
   - BFS and DFS
   - Shortest path algorithms

7. **Hashing**
   - Hash table implementation
   - Collision handling

## Sample Programs

Check the individual files in this directory for complete implementations with explanations.

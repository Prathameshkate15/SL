# Database

This directory contains database connectivity examples and SQL queries commonly covered in practical exams.

## Topics

1. **SQL Basics**
   - CREATE, ALTER, DROP statements
   - INSERT, UPDATE, DELETE operations
   - SELECT queries with WHERE clause

2. **SQL Advanced**
   - Joins (INNER, LEFT, RIGHT, FULL)
   - Subqueries
   - Aggregate functions (COUNT, SUM, AVG, MAX, MIN)
   - GROUP BY and HAVING
   - ORDER BY

3. **Database Connectivity**
   - JDBC (Java Database Connectivity)
   - Connection establishment
   - Executing queries
   - Processing result sets
   - Prepared statements

4. **Practical Examples**
   - Student database management
   - Employee database management
   - Library management system queries

## Sample Programs

Check the individual files in this directory for complete examples with explanations.

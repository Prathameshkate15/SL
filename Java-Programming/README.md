# Java Programming

This directory contains Java programming examples commonly covered in practical exams.

## Topics

1. **Java Basics**
   - Basic syntax
   - Data types and operators
   - Input/Output

2. **OOP in Java**
   - Classes and objects
   - Constructors
   - Inheritance
   - Interfaces
   - Packages

3. **Exception Handling**
   - Try-catch-finally
   - Custom exceptions
   - Throw and throws

4. **Collections Framework**
   - ArrayList
   - HashMap
   - HashSet
   - Iterator

5. **File Handling**
   - Reading from files
   - Writing to files
   - Serialization

6. **Multithreading**
   - Thread creation
   - Synchronization
   - Inter-thread communication

## Sample Programs

Check the individual `.java` files in this directory for complete programs with explanations.
